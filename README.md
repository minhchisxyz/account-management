# Account Management App

This application provides real-time currency exchange rates between EUR and VND, and allows users to add, view, edit, and remove transactions.

## Project Structure & Versions

This repository contains three different implementations/versions of the application:

### 1. Spring Boot & Angular
*   **Folder:** `spring-angular`
*   **Description:** A full-stack application with a Spring Boot backend and an Angular frontend.

### 2. Next.js Fullstack
*   **Folder:** `next`
*   **Description:** A standalone full-stack application built with Next.js.

### 3. Flutter Mobile & Next.js Backend
*   **Folders:** `flutter_mobile` and `next-backend`
*   **Description:** A mobile application built with Flutter (`flutter_mobile`) that communicates with a dedicated API server built with Next.js (`next-backend`). These two projects are designed to work together.
